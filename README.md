
# SearchSearchGo

An alternative UI for Google Search using Official Google APIs.


## Run Locally

Clone the project

```bash
  git clone https://github.com/mayank160920/SearchSearchGo.git
```

Go to the project directory

```bash
  cd SearchSearchGo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## Tech Stack

React, CSS, RapidApi
